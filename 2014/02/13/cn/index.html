新的一年学习规划

2014/13/02 in 计划 using tags

人生当不断奋进。

新的一年开始了。

去年，十分懒惰。

今年，必须勤奋。从今年起，应该患有学习强迫症、早起强迫症、讲卫生强迫症。各种正能量强迫症。

有方向、有方法的勤奋，不是漫无目的、零碎的学习。

每日晚上应该复习一天所学，总结得失。失，必须更改。得，需加以诫勉。

每周日应该回顾一周所学知识，总结得失。并用MD格式挂到自己的个人网站上。

关于娱乐。杜绝上网。除了以下情况：提前完成一天所学、周末休息。只能在周末休息时放电影和音乐。

关于睡眠，应当遵循早睡早起的原则。原则上，晚上12点上床，早上6点半起来。

关于学习效率，断掉一切干扰学习的“障碍物”。比如：kindle、pad、网络、音乐、电影。 只能在图书馆学习。学习知识不建议太杂。比如读论文的同时，就不要学习英语语法来。只干和论文知识相关的事情。学习也不能“贪婪”。

关于制定学习计划，一日有一日的计划、一周有一周的计划、一月得有一月的计划、一个季度有一个季度的计划、半年得有半年的计划、一年得有一年的计划。

关于休息，我制定一条规则把。单周休息半天，双周休息一天，一个月休息两天。

和其他行业的成功相比，学习的成功其实很容易,归结起来就是两点:勤奋和坚持。只要能坚持上面的条条框框，我相信能够博士顺利毕业。加油！！

在论文学习方面，需要学会总结、归纳。阅读完每一篇的论文，写一份规范的学习报告。
