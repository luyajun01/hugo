有三件事值得注意！

2017/09/07 in using tags 生活

最近有三件事觉得自己做的很不好。 第一件事情做的自己觉得自己好蠢呐！

这就是昨天晚上聚餐的时候，一句话就是不懂得察言观色，时刻要记住在这个处室里面，处长是最高权威，任何事情都得围绕着处长来做。那么再回头来看昨晚的事情，当张工说她要回家的时候，你这会儿就不应该说要陪张工回家，正确的做法就是说我要留下来，我得陪乔处吃饭，自己还傻乎乎的和马超说要回家，那么这个就是愚蠢至极了，情商是负值！犯大错了！这时应当大声说出来：我得陪领导吃饭，你得把你的态度亮出来什么态度，就是说你重视领导，你渴望和领导站在一条线上，这才是正确的做法。昨晚，我的反应也说明我平时和领导走的确实不够近，另外一点说明我恐惧和乔处沟通、相处。曾经某位哲人曾经说过，当你越害怕什么东西的时候，你就越要得去尝试着去接触它，不能抗拒，更不能退缩。 第二件事就是个人气质一定要提升！

平时走路一定要挺胸抬头，个人气质很重要，不能畏畏缩缩，更不能猥琐，正常人和其他人拍照的时候一定是挺胸抬头，所以说从明天起一定要在方方面面提醒自己一定要挺胸抬头，做任何事情之前一定要提醒自己挺胸抬头，这个人的气质问题非常非常重要。 第三件事就是最近的工作不饱满不上心，缺乏动力。

我思考了一下为什么缺乏动力呢？我想除了懒惰外，就从是内心里面觉得这些都不是喜欢做这些事情。这也要去克服。尽管说这些事情确实是没有什么意义，但是对于个人来讲，要从细节着手，认认真真地做好每一件事，加快速度，加快效率，扎扎实实的做好每一件事情。
