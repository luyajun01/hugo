一团乱麻

2017/02/12 in using tags 感悟

我发现自己有个很大的问题。就是一个动作下去，常常是坚持不到2周，就立马换了一个另一套想法去干别的去了，就是太随意。

这样做的后果，往往就是学习的东西流于表面，核心内容往往学习不到，最后人脑子的思维都很简单，不深刻。最后整个人就变得很浮躁、不踏实、不自信。。。。

我不想再这么下去了，得想办法解决这个棘手的问题。那么有没有方法呢，我觉得就像走路，得有一个目标，做事得事先制定一个目标，路上还得不断督促自己不忘初心。

我得不断的调整、规范自己的行为，不能由着自己的性子，好好的反思自己的行为！

有三件事值得注意！
