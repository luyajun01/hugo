利用git创建Hugo博客

2017/10/05 in using tags

之前通过Jekyll创建github博客虽然很方便，但是网站编译的速度很慢，为了解决这个问题，现决定把编译网站引擎换成hugo。

具体操作步骤是这样：

下载hugo程序

首先到hugo官网下载hugo.exe

利用git下载博客目录结构

$ cd g:\\hugo\\luyajun01.github.io %在g盘下建立hugo\\luyajun01.github.io文件夹

$ hugo new site luyajun01.github.io %确保hugo文件下有hugo.exe执行文件

然后hugo会自动生成这样一个目录结构：

▸ archetypes/
▸ content/
▸ layouts/
▸ static/
config.toml

接着执行下列命令

$ hugo new post/first.md

执行完后，会在content/post目录自动生成一个MarkDown格式的first.md文件：

+++
date = "2017-05-08T08:36:54-07:00"
draft = true
title = "first"
+++

下面就可以开始写你的文章内容，比如：

+++
date = "2015-01-08T08:36:54-07:00"
draft = true
title = "first"
+++
### Hello Hugo
1. aaa
2. bbb
3. ccc

接下来就是配置hugo主题，我们用谢益辉网站主题，具体操作方法是

$ cd themes %确保主题下载在themes文件下
$ git clone https://github.com/yihui/hugo-lithium-theme.git

安装好之后，就可以编译了

$ hugo server --theme=hugo --buildDrafts --watch

顺便吐槽windows版本的git，竟不能自动粘贴程序，经问度娘后，发现这么解决：

在Bash命令行界面上右键点击左上角，在出现的菜单中点击properties，然后在quickedit上打上对钩。
