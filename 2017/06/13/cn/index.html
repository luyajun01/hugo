2017/13/06 in using tags 阅读

最近读了《浮生六记》。前面的内容读的比较细，因为呢，觉得很有意思。其中，第一章的内容应该来说是这本书最值得读的部分。沈复的妻子芸娘简直就是一位模范妻子，她对沈复的感情令我嫉妒。印象很深的就是有一句话“见余至，必起立偏挪其身”，见微知著，芸娘的人格令人敬仰！第二章“闲情记趣”开头就很熟悉了，小时候读过，此番再读，还是觉得沈复此人还是很有意思的，“夏蚊成雷，私拟作群鹤舞空”，这想象力还是蛮丰富的。余下诸章节，读的很粗，觉得有意思的东西不多，所以就草草翻了翻，便放下了。

在专业书籍方面，上一周研读了关于数据包络分析（Data Envelopment Analysis, DEA）方面的文章，关于其研究内容思路另做详解。

总体来说，还要进一步勤奋！加油！

崔浩死的好！
