构建github page网站

2013/27/11 in 网络技术 using tags 网站

构建github page网站具体步骤:

  • 首先从其他网站clone主题

代码

git clone git@github.com:****/****.github.io.git
  • 在本地建一文件件

代码

cd ****.github.io
  • 然后上传到github网站

代码：

git add .
git commit -m "abc"
git remote set-url origin git@github.com:****/****.github.io.git
git push origin master

如果git push origin master出现问题，可以用下列命令

git push -f origin master
  • 本地预览

代码

jekyll serve

打开 http://localhost:4000/进行本地预览

  • 需要注意的问题

如果在 git remote set-url origin git@github.com:****/****.github.io.git遇到问题，需要执行下列命令 git remote rm origin

_post文件夹下文件名不能是中文的。

关于Git的若干运用实例

  • 如何用命令创建一个名叫code的repository？

touch README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/luyajun01/code.git
git push -u origin master`

  • 如何将一个存在的repository上传到Github上？

git remote add origin https://github.com/luyajun01/code.git
git push -u origin master
